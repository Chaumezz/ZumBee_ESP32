# ZumBee_ESP32
Pequeño bot tipo minisumo, ruedas del Zumo de Pololu, encoders Hall en motores N20, relación 150:1, ESP 32 generico, sensor optico 3 vias OPT3101, IMU Bosch BNO055
Controlado de momento sobre BT Classic ejecutado en el ESP32, un control PID posiciona las ruedas a demanda de la logica de la brujula BNO y el sensor de distancia 3vias OPT.
